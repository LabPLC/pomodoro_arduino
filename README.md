Pomodoro Arduino
================

Reloj pomodoro con un arduino.

Caracteristicas
---------------

* Boton 1/Led azul: Reset
* Boton 2/Led rojo: Modo 25 minutos
* Boton 3/Led verde: Modo 5 minutos
