Pomodoro Arduino
================

Reloj pomodoro con un Arduino.

Características
---------------

* Vibra y suena al terminar el pomodoro
* Botón izquierda/Led azul: Reset
* Botón centro/Led rojo: Modo 25 minutos
* Botón derecha/Led verde: Modo 5 minutos
